# 2.5 社区

Knife4j鼓励开发者通过[获取帮助](help.md)中的方式发起提问，这样有助于Knife4j的社区发展,当然如果你嫌麻烦，也可以直接加入Knife4j提供的QQ群与作者直接交流获取帮助

目前Knife4j有10个QQ技术交流群,群里主要讨论Swagger、Springfox、Knife4j、OpenAPI等项目问题.如果你有兴趣,可以加入其中获取帮助,或者帮助别人.
